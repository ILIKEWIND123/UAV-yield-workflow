| File | Description |

| `RawData.xlsx` | Input dataset containing grain yield, vegetation indices, and NRCT across six wheat growth stages. |

| `Full_workflow.R` | R script for running the full modeling workflow, including GRA-based feature ranking, feature selection, ENR modeling, and fusion-model evaluation. |

| `Feature_selection_error_curves.xlsx` | Training-error records generated when predictors were sequentially added according to GRA ranking. |

| `Full_corrigendum_workflow_results.xlsx` | Integrated workbook containing the main outputs from the complete analysis workflow. |

| `Fusion_comparison_500fold_metrics.xlsx` | Cross-validation performance metrics for multistage feature-level fusion and entropy-weight fusion models. |

| `GRA_rankings.xlsx` | Grey relational analysis rankings for vegetation indices and NRCT at each growth stage. |

| `Selected_features.xlsx` | Final selected predictor subsets for each growth stage and input-data setting. |

| `Thermal_comparison_500fold_metrics.xlsx` | Cross-validation performance metrics comparing vegetation-index-only models with vegetation-index plus NRCT models. |
